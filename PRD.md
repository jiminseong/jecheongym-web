# PRD.md
# 제천 웨이트 전문 헬스장 랜딩 페이지

---

## 1. 프로젝트 개요

### 1.1 목적
- 제천 지역 웨이트 트레이닝 전문 헬스장의 공식 랜딩 페이지 구축
- 네이버 지도 및 검색 유입을 카카오톡 문의로 전환
- “제천 헬스” 키워드에서 광고 없이 상위 노출 가능한 SEO 기반 확보
- 헬스장의 운영 철학과 규칙을 명확히 전달하여 이용자 필터링

### 1.2 레퍼런스
- UI/구조: https://gymboxx.com/gym/29
- 업체 정보: https://map.naver.com/p/entry/place/13146435

---

## 2. 타겟 사용자

### 2.1 핵심 타겟
- 제천 거주 20~40대
- 웨이트 트레이닝 목적이 명확한 사용자
- 자율 운동 환경을 선호하는 사용자

### 2.2 비타겟
- 유산소 위주 운동 목적
- 가벼운 운동, 러닝 목적 사용자
- 규칙·매너에 민감하지 않은 사용자

---

## 3. 핵심 기능 요구사항

### 3.1 가격 정책 및 문의
- 이용권 가격 명확히 공개
- 모든 가격 영역에 카카오톡 문의 버튼 배치
- 카카오톡 채널 연결: `muscle2020`
- 모바일/PC 모두 동작 필수

---

### 3.2 위치 정보
- 네이버 지도 임베드
- 주소 텍스트 표기
- 길찾기 버튼 제공 (네이버 지도 연동)

---

### 3.4 헬스장 기구 소개

#### 요구사항
- 공식 브랜드 이미지 및 설명 사용
- AI 생성 이미지 사용 금지
- 기구별 정보 구성
  - 브랜드명
  - 기구명
  - 공식 설명
  - 이미지
  - 도입일

#### 구현 방식 (초기)
- 정적 JSON 데이터 기반
- 필요 시 공식 홈페이지 자료 수집

### 3.5 최신 기구 업데이트

- 수동 JSON 업데이트
- 월 2~3개 기구 추가 가능 구조

#### 이후 (확장 예정)
- CMS 연동
- 월별 신규 기구 자동 노출
